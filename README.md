# mas-lib
**mas** is a lightweight linear math library for graphics development.

## Features
**mas** provides basic types, utility functions and essential transformations.

**types**

- `Vector2D`
- `Vector3D`
- `Vector4D`
- `Matrix2`
- `Matrix3`
- `Matrix4`

**transformations**
- `scale, translate, rotate`
- `lookat`
- `projection`
