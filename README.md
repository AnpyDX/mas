# Mas-Lib
`mas` is a lightweight, header only C++ math library for graphics.

## Features
### Supported Types
- `Vector2` -> vec2(x, y)
- `Vector3` -> vec3(x, y, z)
- `Vector4` -> vec4(x, y, z, w)
- `Martix2` -> mat2(2x2)
- `Martix3` -> mat3(3x3)
- `Martix4` -> mat4(4x4)

### Supported Transformation
**Model**
- Translate
- Rotate
- Scale

**View**
- LookAt

**Porjection**
- Prepretive
- Ortho
