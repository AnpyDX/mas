# Mas-Lib
`Mas` is a C++ math library for graphics.

## Features
### Supported types
- `Vector2` => (x, y)
- `Vector3` => (x, y, z)
- `Vector4` => (x, y, z, w)
- `Martix2` => (2 x 2)
- `Martix3` => (3 x 3)
- `Martix4` => (4 x 4)

### Provided Transform
- Transform
- Rotation
- Projection
