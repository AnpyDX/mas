# Mas-Lib
`mas` is a lightweight, header only C++ math library for graphics.

## Features
### Supported types
- `Vector2` -> vec2(x, y)
- `Vector3` -> vec3(x, y, z)
- `Vector4` -> vec4(x, y, z, w)
- `Martix2` -> mat2(2 x 2)
- `Martix3` -> mat3(3 x 3)
- `Martix4` -> mat4(4 x 4)

### Supported Transformation
- Translation
- Rotation
- Projection
